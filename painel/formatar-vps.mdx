## Entrando no painel do cliente
1. Faça login em sua conta no **painel do cliente**.
2. Clique em **gerenciar** para acessar o painel de controle do seu VPS.

<img
  className="block"
  src="/images/painel-de-controle.png"
  alt="Painel de controle"
/>

## Escolhendo o sistema operacional

1. Role a página para baixo até encontrar a opção **configurações**.
2. Selecione **Instalações**.
3. Adicione o sistema operacional desejado.
4. Defina uma senha para o VPS.
5. Clique no botão **reinstalar** e aguarde o sistema realizar a formação do VPS.

<img
  className="block"
  src="/images/formatar-vps.png"
  alt="Painel de controle"
/>

<Warning>
É importante lembrar que, ao formatar o VPS, todos os arquivos serão excluídos e um novo sistema operacional será instalado. Certifique-se de fazer um backup de todos os dados importantes antes de prosseguir com a formatação.
</Warning>
