## Formatando o vps no painel de controle
1. Acesse o nosso site e faça o login em sua conta. 
2. Selecione: **Painel do cliente -> Gerenciar** para acessar o painel de controle.

<img
  className="block"
  src="/images/painel-de-controle.png"
  alt="Painel de controle"
/>

## Escolhendo o sistema operacional

1. Role a página para baixo até encontrar o painel.
2. Selecione **Configurações -> Instalações ->  Reinstalar**.
3. Adicione o sistema operacional e crie uma nova senha.
5. Clique no botão **reinstalar** e aguarde o sistema realizar a **formatação do vps**.

<img
  className="block"
  src="/images/vps-formatar.png"
  alt="Painel de controle"
/>

<Warning>
É fundamental lembrar que, ao formatar o vps, todos os arquivos serão apagados e um novo sistema operacional será instalado. Certifique-se de fazer backup de todos os dados importantes antes de prosseguir com a formatação.
</Warning>
