## Abrindo o Prompt de Comando (CMD)
1. Na barra de tarefas, digite **cmd**.
2. Selecione o aplicativo **Prompt de Comando**.

<img
  className="block"
  src="/images/cmd.png"
  alt="Conexão remota"
/>

3. No Prompt de Comando, digite o seguinte comando: `slmgr.vbs -rearm`.
4. Logo em seguida basta reiniciar o seu VPS.

<Check> 
Após o reinício do servidor, a licença do Windows Server será prorrogada por mais 180 dias gratuitos.
</Check>
