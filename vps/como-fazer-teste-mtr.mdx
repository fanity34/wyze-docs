# Baixe o aplicativo

1. Instale o aplicativo [WinMTR](https://app.wyzehost.com.br/WinMTR.exe).
2. Após o download, inicie o aplicativo em seu computador.
3. No campo **host**, insira o endereço IP do seu VPS.
4. Clique no botão **start**  para iniciar o teste.

/>

<Warning>
Aguarde 2 minutos e envie uma imagem para a nossa equipe. O winmtr fornecerá informações detalhadas sobre cada salto de rede até o destino, incluindo latência, perda de pacotes e outras estatísticas relevantes.
</Warning>
